Ansible role which manage sudoers file

#### Variables
```yaml
yum_enabled: yes                               # Enable role
```

#### Usage

Add `Stouts.memcached` to your roles and set vars in your playbook file.

Example:

```yaml

- hosts: all

  roles:
    - Stouts.memcached

```
